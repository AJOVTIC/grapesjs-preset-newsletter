GrapesJS Newsletter Builder is a showcase of what/how is possible to build with the GrapesJS core library. Is not intended to represent a full builder solution.

For any tip about this demo (related to the newsletter construction) check the Newsletter Preset repository and open an issue. For any problem with the builder itself, open an issue on the main GrapesJS repository.

Being a free and open source project contributors and supporters are extremely welcome.
